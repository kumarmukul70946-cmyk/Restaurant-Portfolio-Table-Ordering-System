# 🍽️ Zaika - Premium Restaurant Portfolio & Table Ordering System

![Zaika Cover](https://images.unsplash.com/photo-1517248135467-4c7edcad34c4?auto=format&fit=crop&w=1200&q=80)

Zaika is a sophisticated, full-stack MERN application designed for luxury restaurants. It combines a stunning **Glassmorphism UI** with a seamless **WhatsApp-integrated booking flow** and **direct UPI payment** integration.

---

## ✨ Key Features

- 🥂 **Premium Glassmorphism UI**: A high-end, modern aesthetic using Tailwind CSS v4.
- 📱 **WhatsApp Booking**: Automated reservation confirmation messages sent directly to your customers.
- 💳 **Direct UPI Payments**: Integrated one-tap payment links (PhonePe/GPay/Paytm) for reservation deposits.
- 👤 **Smart User Profiles**: Auto-fills user details and tracks all personal bookings.
- 📊 **Admin Dashboard**: Comprehensive management of restaurants, tables, and reservations.
- 🚀 **Performance Optimized**: Built with Vite and React 19 for lightning-fast interactions.

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/kumarmukul70946-cmyk/Restaurant-Portfolio-Table-Ordering-System.git
cd Restaurant-Portfolio-Table-Ordering-System
```

### 2. Install Dependencies
```bash
npm run install:all
```

### 3. Environment Variables
Create a `.env` file in the `server` directory and add your credentials:
```env
PORT=5000
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
```

### 4. Run Locally
```bash
npm run dev
```
